# 금융 분석 시스템 (Financial Analysis System)

## 소개

이 프로젝트는 다양한 금융 분석 도구를 제공하는 웹 기반 애플리케이션입니다. Flask 백엔드와 Streamlit을 활용하여 다양한 금융 분석 기능을 제공합니다.

## 주요 기능

- 배당 분석 (Dividend Analysis)
- 포트폴리오 최적화 (Portfolio Optimization)
- 퀀텀 분석 (Quantum Analysis)
- 변동성 거래 분석 (Volatility Trading Analysis)
- 기술적 분석 (Technical Analysis)
- 워드 클라우드 분석 (Word Cloud Analysis)

## 시작하기

### 필요 조건

- Python 3.8 이상
- Node.js 14.0 이상
- pip (Python 패키지 관리자)
- npm (Node.js 패키지 관리자)

### 설치 방법

1. 저장소 클론

```bash
git clone [repository-url]
cd FinAnaSys
```

2. Python 의존성 설치

```bash
pip install -r requirements.txt
```

3. Node.js 의존성 설치

```bash
npm install
```

### 실행 방법

애플리케이션 시작:

```bash
npm start
```

이 명령어를 실행하면:

- 자동으로 웹 브라우저가 열립니다
- http://localhost:5050 에서 애플리케이션에 접속할 수 있습니다
